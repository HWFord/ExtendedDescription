• [cat=xx]

Den här taggen infogar ett album i beskrivningen. xx ersätts med albumets unika ID.

  • [photo id=xx album=yy size=M html=yes link=yes]

Med den här taggen kan du infoga en bild av vilken storlek du vill. Inställningar:

- id fotots unika ID
- album (valfritt) albumets unika ID
- size (valfritt) Önskad storlek på bilden, från {SQ, TH, XXS, XS, S, M, L, XL, XXL}
- html (valfritt) Om inställd till no returnerar taggen endast webbadressen till bilden, utan HTML-kod
- link (valfritt) Om inställd till yes blir bilden även en länk till den aktuella bildsidan

  • [random album=yy size=M html=yes link=yes]

Samma som ovan, förutom att fotot väljs slumpmässigt från valt album
