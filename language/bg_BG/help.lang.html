Многоезиково описание (Multilingual descriptions)

Многоезиково описание е поставеното между [lang=xx] and [/lang] тагове, където xx е езиковия код (за пример en, es, fr, bg ...). Има също така два специални езика : [lang=default] и [lang=all]

  • Примери:
    [lang=default] default description [/lang]
    [lang=en] description in english [/lang]
    [lang=de] deutsche Beschreibung [/lang]
    [lang=bg] описание на Български [/lang]

"default" описанието ще бъде използвано по подразбиране там където не е въведено друго описание на езика на който отворил потребител на вашата галерия. Ако [lang=default] не съществува, всичко написано извън тага [lang] ще се приема за описание по подразбиране.

Специален език all: всичко в [lang=all] ще се приеме за описание, БЕЗ да има значение какво е описанието за конкретния език на потребителя на галерията. Това е удобно при използване на HTML/JavaScript код вътре в описанието.
