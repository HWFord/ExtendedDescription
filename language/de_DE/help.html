Ausführliche Beschreibung

Mehrsprachige Beschreibung

Mehrsprachige Beschreibungen werden zwischen [lang=xx] und [/lang] gesetzt, dabei ist xx der Sprachcode (Zum Beispiel en, de, fr). Es gibt auch noch 2 speziell definierte Sprachen : [lang=default] und [lang=all]

  • Beispiel:

    [lang=default] default description [/lang]
    [lang=en] description in english [/lang]
    [lang=de] deutsche Beschreibung [/lang]

Sollte die Beschreibung in der vom Benutzer benützten Sprache nicht vorhanden sein, so wird automatische die standardmässige Beschreibung benützt. Sollte [lang=default] auch nicht vorhanden sein, so wird alles was ausserhalb des [lang] Tags ist als standardmässige Beschreibung benützt.

Spezailsprache all: Alles was sich innerhalb [lang=all] befindet wird in der Beschreibung angezeigt, unabhängig davon in welcher Sprache sich ein Benutzer angemeldet hat. Dies kann nützlich sein um zum Beispiel HTML/Java in einer Beschreibung anzuzeigen.

Ausführlichere Beschreibung

Ausführliche Beschreibung macht es möglich eine Kurz-Beschreibung der Kategorie ausserhalb der Kategorie anzuzeigen, währenddem die ausführlichere Beschreibung der Kategorie innerhalb der Kategorie angezeigt wird, oder aber 2 verschiedene Beschreibungen anzuzeigen.

tag <!--more--> fügt Text zur Kurzbeschreibung hinzu.
tag <!--complete--> gibt dem Benutzer 2 verschiedene Beschreibungen.
tag <!--up-down--> gibt dem Benutzer eine top- und eine bottom-Beschreibung.

  • tag <!--more-->
short description <!--more--> long description

Nur die Kurz-Beschreibung der Kategorie wird ausserhalb der Kategorie angezeigt. In der Kategorie sieht man die Kurz- + ausführliche Beschreibung.

  • tag <!--complete-->
short description <!--complete--> long description

Nur die Kurz-Beschreibung der Kategorie wird ausserhalb der Kategorie angezeigt. In der Kategorie sieht man NUR die ausführliche Beschreibung. So erhalten Sie zwei verschiedene Beschreibungen.

  • tag <!--up-down-->
top description <!--up-down--> bottom description

Nur die Top-Beschreibung der Kategorie wird ausserhalb der Kategorie angezeigt. In der Kategorie wird die Top-Beschreibung oberhalb , und die Bottom-Beschreibung unterhalb der Vignetten angezeigt.

Kategorie oder Foto einfügen

  • tag [cat=xx]

Dieser Tag fügt Kategorien zur Beschreibung hinzu. xx ist als Kategorienamen zu verstehen.

  • tag [img=xx1 xx2 xxN;yyyy;name]

Dieser Tag erlaubt es eine oder mehrere Vignetten in der Beschreibung hinzuzufüggen :

- xx ist das Foto; Sie können auch noch die Kategorie tag [cat=xx] hinzufügen: xx.ccc.
- yyyy (Optional) : weitere unterstützte Befehle {left, right}, zeigen die Beschreibung rechts oder links vom Bild an.

- Name (optional): kann jeden beliebigen Wert annehmen unter (Name, Titelname). Wenn mit Name markiert, wird der Name des Bildes unter der Miniatur angezeigt, mit Titelname wird das Bild Markup <title> mit dem Namen ausgefüllt.

Kategorie verstecken - nicht anzeigen

Dazu benützt man den Tag <!--hidden--> im Kategorienamen:

- diese wird nicht mehr in der Parent-Kategorie angezeigt
- bleibt weiterhin ersichtlich im Kategoriemenü

Kategoriemenü verstecken - nicht anzeigen

Dazu benützt man den Tag <!--mb-hidden--> im Kategorienamen:

- diese wird nicht mehr im Kategoriemenü angezeigt
- bleibt weiterhin ersichtlich in der Parent-Kategorie

Bild verstecken - nicht anzeigen

Dazu benützt man den Tag <!--hidden--> im Bildnamen:

- diese wird nicht mehr auf der Vignettenseite angezeigt
- bleibt weiterhin ersichtlich in der Kategorie

Redirected category

  • [redirect http://piwigo.org]

Insert this tag into a category description to redirect it to the URL of your choice.

  • [redirect img=xx]

Redirect to an image of your gallery where xx is the photo identifier; you can add the category identifier next to the photo identifier: xx.ccc.

  • [redirect cat=xx]

Redirect to a category of your gallery where xx the category identifier.

  • [redirect search=xx]

Redirect to a search page of your gallery where xx is the search identifier.
