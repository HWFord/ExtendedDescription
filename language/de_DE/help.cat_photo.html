• [cat=xx]

Dieser Tag erlaubt das Einfügen eines Albums in der Beschreibung, xx ist die Album ID.

  • [photo id=xx album=yy size=M html=yes link=yes]

Dieser Tag erlaubt das Einfügen eines Bildes mit beliebiger Größe mittels:

- id ID des Bildes
- album (optional) ID des übergeordneten Albums
- size (optional) Größe des Bildes, Auswahl {SQ, TH, XXS, XS, S, M, L, XL, XXL}
- html (optional) bei no liefert der Tag nur nur die URL des Bildes zurück ohne HTML
- link (optional) bei yes wird das Bild mit einem Link zur Bildseite versehen

  • [random album=yy size=M html=yes link=yes]

Genau wie oben, Ausnahme das Bild wird per Zufall aus dem album geholt
