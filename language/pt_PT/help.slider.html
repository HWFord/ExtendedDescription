• [slider album=xx nb_images=10 random=no list=aa,bb,cc size=M speed=3 title=no effect=fade arrows=yes control=yes elastic=no thumbs_size=80]

Esta etiqueta permite inserir um slideshow (NivoSlider), com (você deve definiralbumOR list):

- album(opcional)fonte album
- nb_images(opcional)máximo numero de fotos para exibir no slideshow
- random(opcional) escolhe fotos aleatoriamente no álbum {yes, no}
- list(opcional) uma lista de fotos com ids separados por vírgula
- size(opcional) o tamanho das fotos, entre {SQ, TH, XXS, S, M, L, XL, XXL}
- speed(opcional) Velocidade de transição
- title(opcional) mostra título da foto {yes, no}
- effect(opcional) efeito de transição (opcional) (ver NivoSlider doc)
- arrows(opcional) mostra setas de navegação {yes, no}
- control(opcional) exibe os controles de navegação {yes, no, thumb}
- elastic(opcional) adapta o tamanho do slideshow para cada foto {yes, no}
- thumbs_size(optional) thumbnails size in pixel if control=thumb
