• tag [cat=xx]

Esta etiqueta insere um album na descrição, onde xx é o identificador do album.

  • [photo id=xx album=yy size=M html=yes link=yes]

Esta etiqueta permite inserir uma imagem de qualquer tamanho que você queira, com:

-ididentificador da foto
-album(opcional) o identificador do album pai
-size(opcional) o tamanho da foto, entre {SQ, TH, XXS, S, M, L, XL, XXL}
-html(opcional) se definido para NÃO retorna a URL da foto, sem HTML
-link(opcional) se definido para SIM a foto é incluida num link para a página da foto

  • [random album=yy size=M html=yes link=yes]
