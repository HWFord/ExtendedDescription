Descrições multi-idioma

Descrições multi-idioma são colocados entre etiquetas [lang=xx] e [/lang], onde xx é o código da língua (por exemplo, en, es, fr,pt). Há também 2 linguagens especiais: [lang=default] e [lang=all]

  • Exemplo :

    [lang=default] default description [/lang]
    [lang=en] description in english [/lang]
    [lang=de] deutsche Beschreibung [/lang]
    [lang=pt] descrição português [/lang]

A descrição padrão será usada se a descrição não está disponível no idioma seleccionado pelo utilizador ligado. Se [lang=default] não existe, qualquer coisa fora das etiquetas [lang] será considerada como descrição padrão.

Idioma especial all : nada dentro [lang=all] será incluído na descrição, qualquer que seja o idioma definido pelo utilizador. Isto é principalmente útil para incluir códigos HTML / JavaScript numa descrição.

Descrições extensas

Descrição extensa torna possível ter uma breve descrição para o album fora do album, uma descrição mais longa do album dentro da página do album,ou até 2 descrições distintas dentro da página do album.

tag <!--more--> adiciona texto à descrição curta
. tag <!--complete--> permite que o utilizador tenha 2 descrições distintas.
tag <!--up-down--> permite que o utilizador tenha uma descrição superior e uma descrição inferior

.
  • tag <!--more-->
short description <!--more--> descrição longa

Apenas a descrição curta é exibida fora do album. Na página do album, ver-se-á a descrição curta + a descrição extensa.

  • tag <!--complete-->
short description <!--complete--> long description

apenas a descrição curta é exibida fora do album. Na página do album, ver-se--á a descrição longa, então temos 2 descrições distintas.

  • tag <!--up-down-->
top description <!--up-down--> bottom description

Apenas descrição topo é exibida fora do album. Na página do album, a descrição topo é exibida acima das miniaturas, a descrição inferior é exibida abaixo das miniaturas.

Inserir um album ou uma foto

  • tag [cat=xx]

Esta etiqueta insere um album na descrição, onde xx é o identificador do album.

  • [photo id=xx album=yy size=M html=yes link=yes]

Esta etiqueta permite inserir uma imagem de qualquer tamanho que você queira, com:

- id identificador da foto
- album (opcional) o identificador do album pai
- size (opcional) o tamanho da foto, entre {SQ, TH, XXS, S, M, L, XL, XXL}
- html (opcional) se definido para NÃO retorna a URL da foto, sem HTML
- link (opcional) se definido para SIM a foto é incluida num link para a página da foto

  • [random album=yy size=M html=yes link=yes]

A Mesma coisa que acima, exceto que a foto é escolhida aleatoriamente em random dentro album

  • [slider album=xx nb_images=10 random=no list=aa,bb,cc size=M speed=3 title=no effect=fade arrows=yes control=yes elastic=no]

Esta etiqueta permite inserir um slideshow (NivoSlider), com (você deve definiralbum OR list):

- album (opcional)fonte album
- nb_images (opcional)máximo numero de fotos para exibir no slideshow
- random (opcional) escolhe fotos aleatoriamente no álbum
- list (opcional) uma lista de fotos com ids separados por vírgula
- size (opcional) o tamanho das fotos, entre {SQ, TH, XXS, S, M, L, XL, XXL}
- speed (opcional) Velocidade de transição
- title (opcional) mostra título da foto.
- effect (opcional) efeito de transição (opcional) (ver NivoSlider doc)
- arrows (opcional) mostra setas de navegação
- control (opcional) exibe os controles de navegação
- elastic (opcional) adapta o tamanho do slideshow para cada foto

Hide an album

Use tag <!--hidden--> in the album name:

- it won't be displayed in the parent album
- it remains visible in the album menu

Hide an album from the menu

Use tag <!--mb-hidden--> in the album name:

- it won't be displayed in the album menu
- it remains visible in the parent album

Hide a photo

Use tag <!--hidden--> in the photo name:

- it won't be displayed on the thumbnails page
- it remains visible inside the album

Redirected album

  • [redirect http://piwigo.org]

Insert this tag into a album description to redirect it to the URL of your choice.

  • [redirect img=xx]

Redirect to a photo of your gallery where xx is the photo identifier; you can add the album identifier next to the photo identifier: xx.ccc.

  • [redirect cat=xx]

Redirect to a album of your gallery where xx the album identifier.

  • [redirect search=xx]

Redirect to a search page of your gallery where xx is the search identifier.
