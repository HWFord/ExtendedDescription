Descrições Multi-idioma são colocadas entre as etiquetas [lang = xx] e [/ lang], onde xx é o código da linguagem (por exemplo, en, es, fr, pt). Há também 2 tipos de idioma especiais: [lang = default] e [lang = all] Exemplo: [Lang = default] descrição padrão [/ lang][Lang = en] Descrição em Inglês [/ lang][Lang = de] Descrição em Alemão [/ lang]A descrição padrão será usada se a descrição não está disponível no idioma seleccionado pelo utilizador ligado. Se [lang = default] não existe, qualquer coisa fora das etiquetas [Lang] sera considerados como a descrição padrão. Linguagem especial de tudo: nada dentro das etiquetas [lang = all] será incluído na descrição, independentemente da definição idioma do utilizador ligado. Isto é principalmente útil para incluir ódigos HTML / JavaScript dentro de uma descrição. Descrições extensas Etiquetas (tags) de descrição extensa tornam possível ter uma breve descrição para o album fora do album, uma descrição mais longa para o album dentro da página do album, ou até mesmo duas descrições distintas dentro da página do album. tag adiciona texto a uma curta descrição. tag permitir que o utlizador tenha 2 descrições distintas. tag informa o utilizador que tem uma descrição superior e outra inferior. curta descrição extensa descrição Apenas uma breve descrição é exibida fora do album. Na página do album, ver-se-á a descrição curta + descrição extensa. curta descrição extensa descrição Apenas uma breve descrição é exibida fora do album. Na página do album, a descrição superior surgirá sobre as miniaturas, enquanto que a inferior surgirá sob as miniaturas descrição superior descrição inferior Apenas a descrição superior será exibida fora do album. Na página do album, a descrição do superior é exibida fora do album. Na página do album. a descrição superior é mostrada sobre as miniaturas,a descrição inferior é exibida sob as miniaturas.
