Розширений опис

Багатомовні описи

Багатомовний описи розміщено між [lang=xx] та [/lang] теґами, де xx є мовним кодом (для прикладу en, es, fr, uk). Є також 2 спеціальні мови : [lang=default] and [lang=all]

  • Приклад:

    [lang=default] default description [/lang]
    [lang=en] description in english [/lang]
    [lang=uk] опис на українській [/lang]

Опис за замовчуванням буде використовуватися, якщо опис недоступний в вибраної користувачем мови. Якщо [lang=default] не існує , все, що поза теґом [lang] будуть розглядатися як опис за замовчуванням.

Спеціальні мови всі: все, що всередині [lang=all] будуте включено в опис, все що пов'язане з мовою певного користувача. Це головним чином корисно, щоб включати код HTML/JavaScript усередині опису.

Розширені описи

Розширений опис теґів дозволяє мати короткий опис для альбому за його межами Більш докладний опис для альбому до сторінки альбому або навіть 2 різних описів всередині сторінки альбому.

теґ <!--more--> додає текст в короткий опис.
теґ <!--complete--> дозволяють користувачу мати 2 різних описи.
теґ <!--up-down--> дозволяє користувачу мати верхній і нижній описи.

  • теґ <!--more-->
короткий опис <!--more--> довгий опис

Лише короткий опис відображається за межами альбому. На сторінці альбому ви побачите короткий опис + довгий опис.

  • теґ <!--complete-->
короткий опис <!--complete--> довгий опис

Лише короткий опис відображається за межами альбому. На сторінці альбому відобразиться тільки довгий опис, так що у вас є 2 різні описи.

  • теґ <!--up-down-->
вищий опис <!--up-down--> нижній опис

Тільки вищий опис показується за межами альбому. На сторінці альбому, вищий опис показується вище за згорнуті зображення, нижній опис показується нижче згорнутих зображень.

Вставте альбом або зображення

  • теґ [cat=xx]

Цей теґ вставляє альбом в описі, де хх це ідентифікатор альбому.

  • [photo id=xx album=yy size=M html=yes link=yes]

This tag allows to insert one picture at any size you want, with:

- id the photo identifier
- album (optional) the parent album identifier
- size (optional) the size of the photo, among {SQ, TH, XXS, S, M, L, XL, XXL}
- html (optional) if set to no the tag only returns the URL of the photo, without HTML
- link (optional) if set to yes the photo is wrapped into a link to the picture page

  • [random album=yy size=M html=yes link=yes]

Same thing as above, except that the photo is choosed randomly into album

  • [slider album=xx nb_images=10 random=no list=aa,bb,cc size=M speed=3 title=no effect=fade arrows=yes control=yes elastic=no]

This tag allows to insert a slideshow (NivoSlider), with (you must set album OR list):

- album (optional) source album
- nb_images (optional) max number of photos to display in the slideshow
- random (optional) choose photos randomly in the album
- list (optional) a list of photos ids separated by a comma
- size (optional) the size of the photos, among {SQ, TH, XXS, S, M, L, XL, XXL}
- speed (optional) the slideshow speed (in seconds)
- title (optional) display photo title
- effect (optional) transition effect (see NivoSlider doc)
- arrows (optional) display navigation arrows
- control (optional) display navigation controls
- elastic (optional) adapt slideshow size to each photo

Приховати альбом

Використовуйте теґ <!--hidden--> в назві альбому:

- він не буде відображатися в батьківському альбомі
- він відображається у меню альбому

Приховати альбом у меню

Використовуйте теґ <!--mb-hidden--> в назві альбому:

- він не буде відображатися в меню альбомів
- він залишається видимим у батьківському альбомі

Приховати зображення

Використовуйте теґ <!--hidden--> в назві зображення:

- він не буде відображатися на сторінці мініатюр
- він відображається всередині альбому

Переадресація альбому

  • [redirect http://piwigo.org]

Вставте цей теґ в опис альбому, щоб перенаправити його до URL-адреси на ваш вибір.

  • [redirect img=xx]

Переадресація до зображення галереї, де xx є ідентифікатором зображення; Ви можете додати ідентифікатор альбому поруч із пунктом ідентифікатор зображення: xx.ccc.

  • [redirect cat=xx]

Переадресація до альбому з галереї, де xx ідентифікатор альбому.

  • [redirect search=xx]

Переадресація на сторінку пошуку з галереї, де xx є пошуковим ідентифікатором.
