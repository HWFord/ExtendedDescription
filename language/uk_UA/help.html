Багатомовні описи

Багатомовний описи розміщено між [lang=xx] та [/lang] теґами, де xx є мовним кодом (для прикладу en, es, fr, uk). Є також 2 спеціальні мови : [lang=default] and [lang=all]

  • Приклад:

    [lang=default] default description [/lang]
    [lang=en] description in english [/lang]
    [lang=uk] опис на українській [/lang]

Опис за замовчуванням буде використовуватися, якщо опис недоступний в вибраної користувачем мови. Якщо [lang=default] не існує , все, що поза теґом [lang] будуть розглядатися як опис за замовчуванням.

Спеціальні мови всі: все, що всередині [lang=all] будуте включено в опис, все що пов'язане з мовою певного користувача. Це головним чином корисно, щоб включати код HTML/JavaScript усередині опису.

Розширені описи

Extended description tags make it possible to have a short description for the album outside the album, a longer description for the album inside the album page, or even 2 distinct descriptions inside the album page.

tag <!--more--> adds text to the short description.
tag <!--complete--> let the user have 2 distinct descriptions.
tag <!--up-down--> let the user have a top and a bottom description.

  • tag <!--more-->
short description <!--more--> long description

Only the short description is displayed outside the album. On the album page, you will see short description + long description.

  • tag <!--complete-->
short description <!--complete--> long description

Only the short description is displayed outside the album. On the album page, you will only see the long description, so you have 2 distinct descriptions.

  • tag <!--up-down-->
top description <!--up-down--> bottom description

Only the top description is displayed outside the album. On the album page, the top description is displayed above the thumbnails, the bottom description is displayed below the thumbnails.

Вставте альбом або зображення

  • теґ [cat=xx]

Цей теґ вставляє альбом в описі, де хх це ідентифікатор альбому.

  • теґ [img=xx1 xx2 xxN;yyyy;name]

Цей теґ дозволяє вставляти одну або декілька мініатюр в описі, де:

- xx ідентифікатор зображення; ви можете додати ідентифікатор альбому поряд з ідентифікатором зображення: xx.ccc.
- yyyy (необов'язковий) : може прийняти будь-яке значення між {вліво, вправо}.

- name (необов'язковий) : може прийняти будь-яке значення між {name, titleName}. Якщо теґ, з name назва зображення відображається під слайдами, з titleName розмітку зображення <title> заповнити з ім'ям.

Hide a album

Use tag <!--hidden--> in the album name:

- it won't be displayed in the parent album
- it remains visible in the album menu

Hide a album in menu

Use tag <!--mb-hidden--> in the album name:

- it won't be displayed in the album menu
- it remains visible in the parent album

Hide a photo

Use tag <!--hidden--> in the photo name:

- it won't be displayed on the thumbnails page
- it remains visible inside the album

Переадресація альбому

  • [redirect http://piwigo.org]

Вставте цей теґ в опис альбому, щоб перенаправити його до URL-адреси на ваш вибір.

  • [redirect img=xx]

Переадресація до зображення галереї, де xx є ідентифікатором зображення; Ви можете додати ідентифікатор альбому поруч із пунктом ідентифікатор зображення: xx.ccc.

  • [redirect cat=xx]

Переадресація до альбому з галереї, де xx ідентифікатор альбому.

  • [redirect search=xx]

Переадресація на сторінку пошуку з галереї, де xx є пошуковим ідентифікатором.
