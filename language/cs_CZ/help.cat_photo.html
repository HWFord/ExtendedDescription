• Element [cat=xx]

Tento element vloží kategorii xx do popisu kategorie.

  • [photo id=xx album=yy size=M html=yes link=yes]

Tento element umožní vložit jeden obázek o libovolné velikosti s parametry:

- id identifikátor obrázku
- album (nepovinný) identifikátor nadřazeného alba
- size (nepovinný) velikost obrázku (SQ, TH, XXS, S, M, L, XL, XXL)
- html (nepovinný) pokud ne nastaveno no element pouze vrátí URL obrázku, bez HTML
- link (nepovinný) pokud je nastaveno yes obrázek je zabalen do linku na stránku tohoto obrázku

  • [random album=yy size=M html=yes link=yes]

Stejné jako výše, jen obrázek je vybírán náhodně ze složky album
