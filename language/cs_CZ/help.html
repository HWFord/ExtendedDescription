Vícejazyčné popisy

Vícejazyčné popisy jsou umístěny mezi elementy [lang=xx] a [/lang], kde xx je kód jazyka (například cs, en, es, fr). Dále jsou 2 speciální kódy jazyka: [lang=default] a [lang=all]

  • Příklad:

    [lang=default] default description [/lang]
    [lang=cs] popis v češtině [/lang]
    [lang=en] description in english [/lang]
    [lang=de] deutsche Beschreibung [/lang]

Výchozí popis bude použit v případě, že popis není dostupný v daném jazyce. Pokud [lang=default] neexistuje, tak bude jako výchozí popis použito cokoliv mimo elementy [lang].

Speciální jazyk all: všechno co je uvedeno uvnitř [lang=all] vždy bude vloženo do popisu bez ohledu na zvolený jazyk. Toto je užitečné zejména pro vkládání kódu HTML/JavaScript do popisu.

Rozšířené popisy

Elementy rozšířených popisů umožňují vytvořit zkrácený popis pro kategorii pro použití mimo kategorii a podrobný popis pro použití uvnitř kategorie, nebo dokonce 2 rozdílné popisy pro použití uvnitř kategorie.

Element <!--more--> definuje text pro zkrácený popis.
element <!--complete--> umožňuje vytvořit 2 rozdílné popisy.
element <!--up-down--> umožňuje vytvořit popis do horní a dolní části.

  • Element <!--more-->
krátký popis <!--more--> dlouhý popis

Krátký popis je potom zobrazen mimo kategorii, ale na stránce kategorie je jako popis zobrazen krátký popis + dlouhý popis.

  • Element <!--complete-->
krátký popis <!--complete--> dlouhý popis

Krátký popis je potom zobrazen mimo kategorii, ale na stránce kategorie je jako popis uveden dlouhý popis. Takže pro kategorii existují 2 odlišné popisy.

  • Element <!--up-down-->
horní popis <!--up-down--> dolní popis

Horní popis je potom zobrazen mimo kategorii, ale na stránce kategorie je jako popis zobrazen nahoře horní popis a dole, pod náhledy, dolní popis.

Vložení kategorie nebo fotografie

  • Element [cat=xx]

Tento element vloží kategorii xx do popisu kategorie.

Skrytí kategorie

Je možno využít element <!--hidden--> v názvu kategorie:

- potom nebude viditelná v nadřízené kategorii
- zůstane ale viditelná v seznamu kategorií

Skrytí kategorie v seznamu kategorií

Je možno využít element <!--mb-hidden--> v názvu kategorie:

- potom bude viditelná v nadřízené kategorii
- nebude ale viditelná v seznamu kategorií

Skrytí fotografie

Je možno využít element <!--hidden--> v názvu fotografie:

- fotografie potom nebude viditelná mezi náhledy
- ale zůstane viditelná při prohlížení kategorie

Přesměrování kategorie

  • [redirect http://piwigo.org]

Vložení tohoto tagu do popisu kategorie způsobí přesměrování na URL dle vaší volby.Zde směruje na piwigo.org.

  • [redirect img=xx]

Směrování na obrázek z vaší galerie kde xx je identifikátor ID fotky; také lze přidat ID kategorie vedle ID obrázku: xx.ccc.

  • [redirect cat=xx]

Směruje na danou kategorii ve vaší fotogalerii kde xx je ID kategorie.

  • [redirect search=xx]

Směruje na vyhledávací stránku ve vaší fotogalerii kde xx je ID daného vyhledávání.
