• [slider album=xx nb_images=10 random=no list=aa,bb,cc size=M speed=3 title=no effect=fade arrows=yes control=yes elastic=no thumbs_size=80]

Aquesta etiqueta permet inserir una reproducció de diapositives (NivoSlider), amb (cal establir l'àlbum O llista):

- album (opcional: font de l'àlbum
- nb_images (opcioanl) màxim nombre d'imatges a visualitzar en la reproducció de diapositives de l'àlbum
- random (opcioanl): escull imatges aleatòriament en l'àlbum {yes, no}
- list (opcioanl): una llista de id d'imatges separades per comes
- size (opcional): mida de les imatges, entre {SQ, TH, XXS, S, M, L, XL, XXL}
- speed (opcional): la velocitat de reproducció de diapositives (en segons)
- title (opcional): mostra el títol de la imatge {yes, no}
- effect (opcional): efecte de transició (veure NivoSlider doc)
- arrows (opcional): mostra fletxes de navegació {yes, no}
- control (opcioanl): mostra controls de navegació {yes, no, thumb}
- elastic (opcioanl): s'adapta la reproducció de diapositives a la mida de cada imatge {yes, no}
- thumbs_size (optional) thumbnails size in pixel if control=thumb
