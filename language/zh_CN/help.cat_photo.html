• [cat=xx]

此tag标签可在描述中插入一个相册，xx为相册id。

  • [photo id=xx album=yy size=M html=yes link=yes]

此tag标签可在描述中插入一张任意尺寸的图片：

- id 图片id
- album (可选) 当前相册id
- size (可选) 图片尺寸，有 {SQ, TH, XXS, XS, S, M, L, XL, XXL}
- html (可选) 如果设为 no 则tag标签仅给出图片的URL，不包含HTML
- link (可选) 如果设为 yes 则点击图片可进入该图片所在页面

  • [random album=yy size=M html=yes link=yes]

同上，只是图片是从album中随机抽选的
