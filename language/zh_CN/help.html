扩展描述

多语言描述

多语言描述可放在[lang=xx] 和 [/lang] 标签之间, 其中 xx 是语言代码（如 en,es,cn,fr)。另外还有2个特别的语言: [lang=default] 和 [lang=all]

  • 例如:

    [lang=default] 默认描述 [/lang]
    [lang=en] description in english [/lang]
    [lang=de] deutsche Beschreibung [/lang]

如果用户选择的语言不能使用，将显示默认描述。如果没有 [lang=default] 标签，所有在 [lang] 外的将会以默认描述显示。

特别语言all: 所有在 [lang=all] 标签里的将被包含进描述，不论用户指定了什么语言。这个主要的用处是对于 HTML/JavaScript 等代码在描述里的显示。

扩展描述

扩展描述的标记，使我们能够有一个简短描述在分类外部，同时有一个详细的描述在分类内部页面，或者甚至有2个不同的分类说明在一个分类页面里。

tag <!--more--> 添加简短的描述文本.
tag <!--complete--> 让用户有2种不同的描述.
tag <!--up-down--> 让用户可有一个顶部和底部的描述.

  • tag <!--more-->
简短的描述 <!--more--> 详细描述

只有简短的描述会在分类外显示，在分类页面里，您将看到简短描述 + 详细描述.

  • 标签 <!--complete-->
简短的描述 <!--complete--> 详细描述

只有简短的描述会在分类外显示，在分类页面里，您只能看到详细描述，因此您可以有2种不同的描述.

  • 标签 <!--up-down-->
顶部描述 <!--up-down--> 底部描述

只有顶部描述会在分类外显示，在分类页面里，顶部描述会显示在缩略图上方，底部描述会显示在缩略图的下方.

插入一个分类或相片

  • 标签 [cat=xx]

此标签将把一个分类插入到描述里，xx 即分类的ID.

  • 标签 [img=xx1 xx2 xxN;yyyy;name]

此标签可以在描述里插入一张或者多张缩略图:

- xx 是相片的ID; 您可以在相片ID旁边添加分类的ID: xx.ccc.
- yyyy (optional) : 可输入任何值，如：{left, right}.

- name (optional) : 可输入任何值，如：{name, titleName}. 如果填写了此项为 name ，那么将在缩略图下显示此相片的名称，如果填写为 titleName ,相片里的 <title> 标记里的内容将会替换了名称的内容.

隐藏一个分类

在分类名称里使用标签 <!--hidden--> :

- 它将不会在父分类中显示
- 但它仍可在分类菜单中看见

- 它将不会在分类菜单中显示
- 但它仍可在父级分类中看见
