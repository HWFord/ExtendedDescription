Ši žyma leidžia įterpti betkokio dydžio paveiksiuką, kur:

- id nuotraukos indentifikatorius
- album (neprivaloma) pagrindinio albumo identifikatorius
- size (neprivaloma) nuotraukos dydis, nurodomas: {SQ, TH, XXS, XS, S, M, L, XL, XXL}
- html (neprivaloma) jei nustatyta į no žyma grąžina tik nuotraukos nuoroda, be HTML
- link (neprivaloma) jei nustatyta į yes nuotrauka įpinta į paveiksliuko puslapio nuorodą

  • [random album=yy size=M html=yes link=yes]
