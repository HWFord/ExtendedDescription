Multilanguage beschrijvingen

Multilanguage beschrijvingen worden geplaatst tussen [lang=xx] en [/lang] tags, waar xx de taalcode is (bv: en, es, fr). Er zijn ook 2 speciale talen: [lang=default] en [lang=all]

  • Voorbeeld:

    [lang=default] default description [/lang]
    [lang=en] description in english [/lang]
    [lang=de] deutsche Beschreibung [/lang]

De standaard omschrijving zal gebruikt worden als de beschrijving niet beschikbaar is in de voor de huidige gebruiker geselecteerde taal. Als [lang = default] niet bestaat, zal alles buiten de [lang] tags worden beschouwd als de standaard omschrijving.

Special language all: alles binnen [lang=all] zal worden ingevoegd ,ongeacht wat de taal van de gebruiker is. Dit is handig voor het invoegen van HTML/JavaScript code binnen een omschijving.

Uitgebreide omschrijvingen

Uitgebreide omschrijving tags maken het mogelijk om een korte omschrijving te maken voor eht album buiten het album, en een langere omschrijving of zelfs 2 verschillende omschrijvingen binnen de album pagina.

tag <!--more--> tekst voor de korte omschrijving.
tag <!--complete--> laat de gebruiker 2 verschillende omschrijvingen.
tag <!--up-down--> laat de gebruiker een kop en voet omschrijving.

  • tag <!--more-->
korte omschrijving <!--more--> lange omschrijving

Alleen de korte omschijving word getoond buiten het album. Op de album pagina, zie je een korte en lange omschrijving.

  • tag <!--complete-->
korte omschrijving <!--complete--> lange omschrijving

>Alleen de korte omschijving word getoond buiten het album. Op de album pagina, zie je alleen de lange omschrijving. Zo heb je 2 verschillende omschrijvingen.

  • tag <!--up-down-->
top description <!--up-down--> bottom description

Alleen de top omschrijving is zichtbaar buiten het album. Op de album pagina, de kop omschrijving wordt getoond boven de miniaturen, de voet omschrijving wordt getoond onder de miniaturen.

Invoegen van foto of album

  • label [cat=xx]

Deze label voegt een album in de omschrijving in, waar xx is het album ID.

Verberg een album

Gebruik het label <!--hidden--> in de album naam:

- het wordt niet getoond in het bovenliggende album
- het blijft zichtbaar in het album menu

Gebruik het label <!--mb-hidden--> in de album naam:

- het wordt niet getoond in het album menu
- het blijft zichtbaar in het bovenliggende album

Verberg een foto

Gebruik het label <!--hidden--> in de foto naam:

- het wordt niet getoond op de miniaturen pagina
- het blijft zichtbaar binnen het album
