• [cat=xx]

Dit label voegt een album toe in de beschrijving, waarbij xx het ID van het album is.

  • [photo id=xx album=yy size=M html=yes link=yes]

Met dit label kunt u een afbeelding van zelfbepaalde grootte toevoegen, met:

- id het foto-nummer
- album (optioneel) het nummer van het hoofdalbum identifier
- size (optioneel) de grootte van de foto, van {SQ, TH, XXS, XS, S, M, L, XL, XXL}
- html (optioneel) indien ingesteld op no zal dit label alleen de URL van de foto opleveren, zonder HTML
- link (optioneel) indien ingesteld op yes zal de foto middels een link op de fotopagina worden toegevoegd

  • [random album=yy size=M html=yes link=yes]
