Descrições em vários idiomas são colocados entre [lang=xx] e [/lang] etiquetas, onde xx é o código de idioma (por exemplo en, es, fr). Há também dois idiomas especiais : [lang=default] e [lang=all]

  • Examplo:
    [lang=default] descrição padrão [/lang]
    [lang=en] descrição em Inglês [/lang]
    [lang=de] descrição em alemão [/lang]

A descrição padrão será utilizado se a descrição selecionada não estiver disponível no idioma do usuário conectado. Se [lang=default] não existir, qualquer coisa fora da [lang] tags será considerada como a descrição padrão.

Linguagem especial all: qualquer coisa dentro [lang=all] será incluído na descrição, qualquer que seja o idioma definido para o usuário conectado. Isso é útil principalmente para incluir código HTML/JavaScript dentro de uma descrição.
