Etiquetas de descrição Extendidas torna possível ter uma breve descrição para o álbum fora do álbum, uma descrição mais longa para o álbum dentro da página do álbum, ou até 2 Descrições distintas dentro da página de álbum

etiqueta <!--more--> acrescenta o texto para a descrição curta.
etiqueta <!--complete--> permite que o usuário tenha 2 descrições distintas.
etiqueta <!--up-down--> permite ao usuário ter um top e uma descrição em baixo.

descrição curta <!--complete--> descrição longa

Apenas a descrição curta é exibida fora do álbum. Na página do álbum, você só vai ver a descrição longa, então você tem duas descrições distintas.

descrição no topo <!--up-down--> descrição em baixo

Apenas a descrição superior é exibida fora do álbum. Na página do álbum, a descrição superior é exibido acima das miniaturas, a descrição inferior é exibida abaixo das miniaturas.
