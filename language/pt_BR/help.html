Descrição Extendida

Descrições em vários idiomas

Descrições em vários idiomas são colocados entre [lang=xx] e [/lang] etiquetas, onde xx é o código de idioma (por exemplo en, es, fr). Há também dois idiomas especiais : [lang=default] e [lang=all]

  • Examplo:
    [lang=default] descrição padrão [/lang]
    [lang=en] descrição em Inglês [/lang]
    [lang=de] descrição em alemão [/lang]

A descrição padrão será utilizado se a descrição selecionada não estiver disponível no idioma do usuário conectado. Se [lang=default] não existir, qualquer coisa fora da [lang] tags será considerada como a descrição padrão.

Linguagem especial all: qualquer coisa dentro [lang=all] será incluído na descrição, qualquer que seja o idioma definido para o usuário conectado. Isso é útil principalmente para incluir código HTML/JavaScript dentro de uma descrição.

Descrições Extendidas

Etiquetas de descrição Extendidas torna possível ter uma breve descrição para o álbum fora do álbum, uma descrição mais longa para o álbum dentro da página do álbum, ou até 2 Descrições distintas dentro da página de álbum

etiqueta <!--more--> acrescenta o texto para a descrição curta.
etiqueta <!--complete--> permite que o usuário tenha 2 descrições distintas.
etiqueta <!--up-down--> permite ao usuário ter um top e uma descrição em baixo.

  • <!--more-->
descrição curta <!--more--> descrição longa

Apenas a descrição curta é exibida fora do álbum. Na página do álbum, você vai ver breve descrição + descrição longa.

  • <!--complete-->
descrição curta <!--complete--> descrição longa

Apenas a descrição curta é exibida fora do álbum. Na página do álbum, você só vai ver a descrição longa, então você tem duas descrições distintas.

  • <!--up-down-->
descrição no topo <!--up-down--> descrição em baixo

Apenas a descrição superior é exibida fora do álbum. Na página do álbum, a descrição superior é exibido acima das miniaturas, a descrição inferior é exibida abaixo das miniaturas.

Insira um álbum ou uma foto

  • [cat=xx]

Issa etiqueta insere um álbum na descrição, onde xx é o identificador do álbum.

  • [foto id=xx album=yy tamanho=M html=yes link=yes]

Essa etiqueta permite que você insira uma imagem de qualquer tamanho que você quiser, com:

- id o identificador de foto
- album (opcional) o identificador de álbum pai
- size (opcional) o tamanho da fotografia, de {SQ, TH, XXS, XS, S, M, L, XL, XXL}
- html (opcional) se configurado para não a etiqueta só retorna a URL da foto, sem HTML
- link (opcional) se configurado para sim a foto está empacotada em um link para a página imagem

  • [álbum aleatório=yy tamanho=M html=sim link=sim]

A mesma coisa como acima, exceto que a foto é escolhida aleatoriamente a partir do album

  • [slider album=xx nb_imagens=10 aleatório=não lista=aa,bb,cc tamanho=M velocidade=3 título=no efeito=desvanecer setas=sim controle=sim elástica=não]

Essa etiqueta permite inserir um slideshow (NivoSlider). (voce tem que configurar album OU lista):

- album (opcional) album fonte
- nb_images (opcional) número máximo de fotos para exibir no slideshow
- random (opcional) escolher aleatoriamente fotos no álbum {yes, no}
- list (opcional) uma lista de fotos ids separados por uma vírgula
- size (opcional) o tamanho das imagens, de {SQ, TH, XXS, XS, S, M, L, XL, XXL}
- speed (opcional) a velocidade da apresentação (em segundos)
- title (opcional) exibir o título da foto {yes, no}
- effect (opcional) efeito de transição (ver NivoSlider doc)
- arrows (opcional) exibir setas de navegação {yes, no}
- control (opcional) exibir controles de navegação {yes, no, thumb}
- elastic (opcional) adaptar tamanho do slideshow para cada foto {yes, no}
- thumbs_size (optional) thumbnails size in pixel if control=thumb

Esconder um álbum

Use tag <!--hidden--> no nome do álbum:

- não será exibido no álbum pai
- mantém-se visível no menu de álbum

Esconder um álbum a partir do menu

Use tag <!--mb-hidden--> no nome do álbum:

- não será exibida no menu álbum
- mantém-se visível no álbum pai

Esconder uma foto

Use tag <!--hidden--> no nome da foto:

- será exibida na página miniaturas
- mantém-se visível no interior do álbum

Álbum redirecionado

  • [redirecionar http://piwigo.org]

Inserir esta etiqueta na descrição do álbum para redirecioná-lo para o URL de sua escolha.

  • [redirecionar img=xx]

Redirecionar para uma foto de sua galeria onde xx é o identificador da foto, você pode adicionar o identificador do álbum ao lado do identificador da foto: xx.ccc.

  • [redirecionar cat=xx]

Redirecionar para um álbum de sua galeria onde xx o identificador de álbum.

  • [redirecionar search=xx]

Redirecionar para uma página de busca de sua galeria onde xx é o identificador de busca.
