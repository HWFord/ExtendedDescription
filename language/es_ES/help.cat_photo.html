• [cat=xx]

Esta etiqueta se inserta un álbum en la descripción, donde xx es el identificador del álbum.

  • [photo id=xx album=yy size=M html=yes link=yes]

Esta etiqueta le permite insertar una imagen de cualquier tamaño que desee, con:

- id el identificador de la foto
- album (opcional) el identificador del álbum padre
- size (optional) el tamaño de la foto, desde {SQ, TH, XXS, XS, S, M, L, XL, XXL}
- html (optional) if set to no (opcional) si se establece en no la etiqueta sólo devuelve la dirección URL de la foto, sin HTML
- link (opcional) si se establece en sí la foto estará envuelta en un enlace a la página de la imagen

  • [random album=yy size=M html=yes link=yes]

Lo mismo que el anterior, excepto que la foto es elegida al azar album
