• [slider album=xx nb_images=10 random=no list=aa,bb,cc size=M speed=3 title=no effect=fade arrows=yes control=yes elastic=no thumbs_size=80]

Этот тэг позволяет Вам создать слайд-шоу(основано на NivoSlider). (Вы должны указать album ИЛИ list):

- album (не обязательно) - альбом-источник изображений
- nb_images (не обязательно) - максимальное число изображений, показываемых в слайд-шоу
- random (не обязательно) - случайный выбор изображений из альбома {yes, no}
- list (не обязательно) - список идентификаторов изображений (должны быть разделены запятыми)
- size (не обязательно) - размеры изображений, из диапазона {SQ, TH, XXS, XS, S, M, L, XL, XXL}
- speed (не обязательно) - скорость смены изображений (в секундах)
- title (не обязательно) - отображать название изображений {yes, no}
- effect (не обязательно) - transition effect (подробности в NivoSlider doc)
- arrows (не обязательно) - отображать стрелки навигации {yes, no}
- control (не обязательно) - отображать панель навигации {yes, no, thumb}
- elastic (не обязательно) - адаптировать размеры слайд-шоу для каждого изображения {yes, no}
- thumbs_size (не обязательно) - размеры миниатюр (в пикселях) - если ранее была включена функция control=thumb
