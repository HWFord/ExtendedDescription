Тег расширенного описания позволяет иметь короткое описание для альбома вне альбома, длинное описание альбома внутри страницы альбома, или даже 2 отдельных описания внутри страницы альбома.

тег <!--more--> добавляет текст к короткому описанию.
тег <!--complete--> позволяет пользователю иметь два отдельных описания.
тег <!--up-down--> позволяет пользователю иметь верхнее и нижнее описания.

  • тег <!--more-->
короткое описание <!--more--> длинное описание

Только короткое описание будет отображаться вне альбома. На странице альбома Вы увидите короткое описание + длинное описание.

  • тег <!--complete-->
короткое описание <!--complete--> длинное описание

Только короткое описание будет отображаться вне альбома. На странице альбома Вы увидите только длинное описание, таким образом у Вас будет два отдельных описания.

  • тег <!--up-down-->
верхнее описание <!--up-down--> нижнее описание

Только короткое описание будет отображаться вне альбома. На странице альбома верхнее описание будет отображаться над миниатюрами, а нижнее, соответственно под ними.
