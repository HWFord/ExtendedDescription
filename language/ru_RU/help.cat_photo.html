• [cat=xx]

Этот тэг вставляет альбом в описание, при этом xx - это идентификатор альбома.

  • [photo id=xx album=yy size=M html=yes link=yes]

Этот тэг позволяет Вам вставить одно изображение любого размера, который пожелаете. При этом:

- id - идентификатор изображения
- album (не обязательно) - идентификатор альбома, содаржещего нужное изображение
- size (не обязательно) - размер изображения, из диапазона {SQ, TH, XXS, XS, S, M, L, XL, XXL}
- html (не обязательно)- если указать no, то тэг указывает только URL изображения, без HTML
- link (не обязательно)- если указать yes, то изображение становится ссылкой на картинку

  • [random album=yy size=M html=yes link=yes]

То же самое, что и выше, но только изображение выбирается случайным образом из альбома album
