Мультиязычные описания располагаются между тегами [lang=xx] и [/lang], где xx - код языка (например ru, en, es, fr). Также имеются 2 специальных языка: [lang=default] и [lang=all]

  • Пример:
    [lang=default] описание по умолчанию [/lang]
    [lang=ru] описание на русском [/lang]
    [lang=de] deutsche Beschreibung [/lang]

Описание по умолчанию будет использовано в случае, если отсутствует описание на выбранном текущим пользователем языке. Если [lang=default] не существует, все снаружи тега [lang] будет считаться описанием по умолчанию.

Специальный язык all: все внутри тега [lang=all] будет включено в описание независимо от выбранного текущим пользователем языка. Чаще всего это используется для включения HTML/JavaScript кода в описание.
