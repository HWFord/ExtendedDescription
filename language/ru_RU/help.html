Мультиязычные описания

Мультиязычные описания располагаются между тегами [lang=xx] и [/lang], где xx - код языка (например ru, en, es, fr). Также имеются 2 специальных языка: [lang=default] и [lang=all]

  • Пример:

    [lang=default] описание по умолчанию [/lang]
    [lang=ru] описание на русском [/lang]
    [lang=de] deutsche Beschreibung [/lang]

Описание по умолчанию будет использовано в случае если отсутствует описание на выбранном текущим пользователем языке. Если [lang=default] не существует, все снаружи тега [lang] будет считаться описание по умолчанию.

Специальный язык all: все внутри тега [lang=all] будет включено в описание независимо от выбранного текущим пользователем языка. Чаще всего это используется для включения HTML/JavaScript кода в описание.

Расширенное описание

Тег расширенного описания позволяет иметь короткое описание для альбома вне альбома, длинное описание альбома внутри страницы альбома, или даже 2 отдельных описания внутри страницы альбома.

тег <!--more--> добавляет текст к короткому описанию.
тег <!--complete--> позволяет пользователю иметь два отдельных описания.
тег <!--up-down--> позволяет пользователю иметь верхнее и нижнее описания.

  • тег <!--more-->
короткое описание <!--more--> длинное описание

Только короткое описание будет отображаться вне альбома. На странице альбома Вы увидите короткое описание + длинное описание.

  • тег <!--complete-->
короткое описание <!--complete--> длинное описание

Только короткое описание будет отображаться вне альбома. На странице альбома Вы увидите только длинное описание, таким образом у Вас будет два отдельных описания.

  • тег <!--up-down-->
верхнее описание <!--up-down--> нижнее описание

Только короткое описание будет отображаться вне альбома. На странице альбома верхнее описание будет отображаться над миниатюрами, а нижнее, соответственно под ними.

Вставка альбома либо изображения

  • тег [cat=xx]

Этот тег вставляет альбом в в описание, где xx идентификатор альбома.

  • тег [img=xx1 xx2 xxN;yyyy;название]

Этот тег позволяет Вам вставлять одну или больше миниатюр в описание, где:

- xx идентификатор изображения; Вы можете добавить идентификатор альбома за идентификатором изображения: xx.ccc.
- yyyy (необязательно) : может принимать значения {left, right}.

- название (необязательно) : может принимать значения {name, titleName}. Если добавлено название оно будет отображено под миниатюрой, либо если titleName поле фотографии <title> будет заменено на название.

Скрытие альбома

Используйте тег <!--hidden--> в имени альбома:

- он не будет отображен в родительском альбоме
- от останется видимым в меню альбома

Скрытие альбом в меню

Используйте тег <!--mb-hidden--> в имени альбома:

- он будет отображен в родительском альбоме
- от не останется видимым в меню альбома

Скрытие фотографии

Используйте тег <!--hidden--> в названии изображения:

- оно не будет отображено на станице миниатюр
- оно останется видимым в альбоме

Перенаправленные альбомы

  • [redirect http://piwigo.org]

Используйте этот тег в описании альбома для перенаправления его на выбранный URL.

  • [redirect img=xx]

Перенаправление на изображение в Вашей галерее, где xx - идентификатор фотографии; Вы можете добавить идентификатор альбома за идентификатором изображения: xx.ccc.

  • [redirect cat=xx]

Перенаправление на альбом в Вашей галерее, где xx - идентификатор альбома.

  • [redirect search=xx]

Перенаправление на поисковую страницу Вашей галереи, где xx - поисковый идентификатор.
