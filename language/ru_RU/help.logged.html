• [login-link html=true text="log in[lang=ru]Вход по логину[/lang]"]

Этот признак позволяет устанавливать автоматическую связь логина с текущей страницей.

- html (дополнительно) если Вы установите ключ false, то будет вызываться только URL для логина без HTML
- text (дополнительно) ссылка, которая может содержать признак языка [lang]

  • [logged]

Отображать блок текста только текущему пользователю или нет.

Например:
[logged] С возвращением [/logged]
[logged=false] Пожалуйста, введите логин [/logged]
